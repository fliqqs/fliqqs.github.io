So this function iterates over one letter at a time ensuring each one is correct. We take our inputed character and have to check that it matches the value of XOR scamble, mystery. Mystery is a value stored in the binary and scambled being an operation run on the current letter index.

Lets follow along just with the first letter out of curiosity. I will list each dissasembled instruction and its asm.

0x556f3fe82329                  movzx  ebx, BYTE PTR [rax+rdx*1]

flag_current_char = (&DAT_00104020)[local_3c];

│gef➤  x/1b $rax + $rdx

So the scramble function given 0 returns 0xffffffaa we note that if we xor the first byte with the 0xcc we get 0x66 which is an ascii f. The start of flag, if we go a little further in the instructions we can just check the comparision.

if (input[local_3c] != (scramble ^ flag_current_char)) {
0x556f3fe82354                  cmp    dl, al

We can also see here that al holds 0x66, so we can build up the flag one byte at a time to get the result.

We can follow along in gdb and see what is going to be loaded

I found a really cool article describing the same problem I was having with GDB being able to automate debugging output of registers.

https://sdimitro.github.io/post/scripting-gdb/

This allowed me to run a python program that would automate the test input hit a breakpoint at the comparision for that letter and read the registers and append it to the payload. Doing this incrementally allowed for building of the flag. Here was my final solution it did require being re-ran a few times to get the flag but

from pwn import *
context.log_level = 'DEBUG'
# context(terminal=['tmux', 'splitw', '-h'])
context.binary = './crackme2'
elf = context.binary

p = process(['stdbuf', '-o0', './crackme2'])

p.recvuntil(b'Enter flag:')

pid = p.pid
with open(f'/proc/{pid}/maps', 'r') as f:
    for line in f:
        if elf.path in line and 'r-xp' in line:
            base_addr = int(line.split('-')[0], 16)
            break

target_addr = base_addr + 0x354  # Offset to cmp inst
log.info(f"Base address: {hex(base_addr)}")
log.info(f"Target address: {hex(target_addr)}")


output = gdb.attach(p, gdbscript=f"""
    set pagination off
    set logging file gdb.output
    set logging on
    break *{hex(target_addr)}
        command 1
        printf "AL: %02x\\n", $al
        continue
    end
    continue
""", api=True)

time.sleep(3)

p.sendline(b'B' * 16)
leak = p.wait_for_close()

print(output)
